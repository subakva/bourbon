bourbon by subakva

Liquid template support for Sinatra

Bourbon is an extension to the Sinatra web application framework which integrates the Liquid template engine into the standard Sinatra template rendering system.

Dependencies

  • Sinatra v.0.9.1
  • Liquid 2.0

Install

For a "classic" Sinatra app, just add: 

require 'sinatra/bourbon'

For a "modular" Sinatra app: 

require 'sinatra/bourbon'
class LiquidApp < Sinatra::Base
  helpers Sinatra::Bourbon
end
